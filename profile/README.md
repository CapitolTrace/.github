### What we build

Capitol Trace aggregates **25+ public government data sources** into connected, searchable intelligence that any citizen, journalist, or researcher can use for free. We track the current congressional roster — how they vote, who funds them, what they trade, who lobbies them — and surface patterns that no single database reveals.

### By the numbers

| | |
|:--|:--|
| **25+ data sources** | Congress.gov, FEC, Senate LDA, CISA, ACLED, CelesTrak, GDELT, and more |
| **538 member profiles** | Every sitting member of Congress with 17 data sections per dossier |
| **96,000+ stock trades** | Congressional trading monitor with STOCK Act compliance scoring |
| **187 countries monitored** | Real-time NatSec dashboard with 7 intelligence feeds |
| **9 live API endpoints** | Normalized data for developers, newsrooms, and researchers |
| **12 MCP tools** | AI-native access for Claude, GPT, and Gemini |
